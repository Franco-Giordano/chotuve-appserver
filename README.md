## Instrucciones

1. Instalar [Docker Engine](https://docs.docker.com/engine/install/) y [Docker Compose](https://docs.docker.com/compose/install/)

2. Levantar server + database: `docker-compose up`

4. Probar la REST API en `0.0.0.0:5000`
